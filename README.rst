Install
------------------------------------------------------------------------------

``aws_dynamodb_io`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install aws-dynamodb-io

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws-dynamodb-io
